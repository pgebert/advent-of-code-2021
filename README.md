# advent-of-code-2021
Advent of Code 2021 solutions written in Python.

## Getting started

(Optional) Create a conda environment:
```
conda create -n adventOfCode2021
conda activate adventOfCode2021
```

Install local package:

```
cd advent-of-code-2021
pip install -e .
```

Run tests:

```
pytest
```
